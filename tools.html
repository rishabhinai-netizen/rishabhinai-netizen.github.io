Build Philosophy

Why I Build

Gap-First Thinking
Every tool I build addresses a gap that no commercial product covers — specifically in Indian HFC fraud operations and NSE-specific trading.
Offline-First for BFSI
Internal fraud tools are single-file HTML with zero server dependency — so investigation teams can use them in air-gapped or restricted environments.
AI as Force Multiplier
I use Claude, Groq, and GitHub Actions not for novelty but to remove real friction — from STR drafting to concall parsing to Telegram alerting at market open.
